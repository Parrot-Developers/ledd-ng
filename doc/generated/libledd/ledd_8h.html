int ledd_init_impl ( const char *  global_config,
bool  skip_plugins 
)

Initializes the ledd server.

Parameters
global_configPath to the global configuration file, usually /etc/ledd/global.conf
skip_pluginsif false, the plugins dir (defined in global.conf, usually set to "/usr/lib/ledd-plugins/"), will be scanned and the plug-ins found will be loaded, if false, no plug-in wil be loaded. This is useful if the plug-ins you want to use are already linked into your program, like when using the libledd-static version of libledd
Returns
errno-compatible negative value on error, 0 on success
Note
in most case, one should rather use directly ledd_init() directly
